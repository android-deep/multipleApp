# 简介
本项目基于济宁市罗盒网络科技有限公司2017年12月已停止更新的开源代码VirtualApp，以及第三方作者tinycode1的VirtualBox开源项目进行开发。  
本项目发布符合GPL3.0协议及任何符合第7条的附加条件
